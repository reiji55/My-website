# 実技タイマー PWA 一式

このフォルダの中身をそのまま Web に置くと、インストール可能な PWA（アプリ）になります。

## 含まれるファイル
- `index.html` … アプリ本体
- `manifest.webmanifest` … アプリ名・アイコン・表示設定
- `sw.js` … オフライン動作＆更新制御（サービスワーカー）
- `icon-192.png` / `icon-512.png` / `icon-512-maskable.png` … Android/一般用アイコン
- `apple-touch-icon.png` … iPhone/iPad 用アイコン

**5つのファイルは必ず同じ階層に、この名前のまま置いてください**（名前を変えると読み込めません）。

## GitHub Pages への置き方
1. リポジトリに上記ファイルを全部アップロード（`index.html` はルート直下）。
2. Settings → Pages で公開。数分で `https://<ユーザー名>.github.io/<リポジトリ名>/` で開けます。
3. スマホの Safari / Chrome でそのURLを開く。

## アプリとしてインストール
- **iPhone / iPad（Safari）**: 共有ボタン →「ホーム画面に追加」。
- **Android（Chrome）**: メニュー →「アプリをインストール」。

インストールすると全画面のアプリとして起動し、オフラインでも動きます。

## APK（Androidアプリ）にする
1. 上記のとおり公開して、URL を用意する。
2. https://www.pwabuilder.com/ を開き、そのURLを入力。
3. 「Android」→ パッケージ生成 → APK（署名付き）をダウンロード。
   - 自分・友人の端末に入れるだけなら、生成された APK をそのまま端末に転送してインストールでOK。
   - Google Play に出す場合は AAB を使い、デベロッパー登録が必要です。

## アプリを更新したいとき
`index.html` を差し替え、`sw.js` の先頭 `const VERSION = 'jitsugi-timer-v1';` の番号を
`v2`, `v3`... と上げて一緒にアップロードしてください。
利用者が次にオンラインで開いたとき、自動で新しい版に入れ替わります。

## データについて
記録（練習履歴・メモ・設定）は各利用者の端末内に保存されます。
アプリ本体を更新しても、この保存データは消えません。
